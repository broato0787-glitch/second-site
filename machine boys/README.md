# Swedru Secondary School 23rd Instrumentalist Website

A modern static website for the Swedru Secondary School 23rd Instrumentalist event.

## Features
- Responsive modern layout
- Homepage hero, event highlights, countdown timer, schedule, performers, testimonials, and newsletter signup
- About page with mission and leadership summary
- Program page with schedule and FAQ
- Contact page with contact cards and inquiry form
- Mobile-friendly navigation menu

## Assets
- Place the school logo as `images/swesco.jpg`

## How to View
Open `index.html` in a web browser.

## Technologies
- HTML
- CSS
- JavaScript
